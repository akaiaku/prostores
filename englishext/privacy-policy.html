Privacy Policy

English Learning Assistant  ·  Last updated: April 8, 2026

TL;DR This extension does not collect, store, or share any personal data. Your vocabulary list lives entirely on your own device.

1. Information We Collect

English Learning Assistant does not collect any personally identifiable information. The extension does not require an account, login, or registration.

The only data stored by this extension is:

  • Your saved vocabulary words (word text, definition, phonetic, audio URL, SRS level, review dates)
  • Your selected translation language preference
  • A one-time flag indicating whether the onboarding screen has been seen

All of the above is stored locally on your device using chrome.storage.local and never leaves your browser.

2. External API Calls

To provide its core features, the extension makes requests to two third-party APIs only when you highlight a word:

  • Free Dictionary API (api.dictionaryapi.dev) — The highlighted word is sent as a URL path parameter to retrieve its definition, phonetic transcription, and audio pronunciation URL.
  • MyMemory Translation API (api.mymemory.translated.net) — The highlighted word and your chosen target language code are sent as query parameters to retrieve a translation.

Only the word being looked up is transmitted. No page content, browsing history, user identity, or device information is included in these requests. Please review the respective privacy policies of these services for how they handle API requests.

3. Data Sharing

We do not sell, trade, rent, or otherwise share any user data with third parties. No analytics, advertising, or tracking services are used.

4. Data Security

All vocabulary data is stored locally using Chrome's built-in chrome.storage.local API, which is sandboxed to this extension. No external servers store your vocabulary list.

5. Children's Privacy

This extension does not knowingly collect any information from children under 13. It contains no age-gating features as no personal data is collected from any user.

6. Changes to This Policy

If this privacy policy is updated, the new version will be published at this URL and the "Last updated" date at the top will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.
